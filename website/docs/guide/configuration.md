## Environment Variables

- `CARROT_API_KEY` - Your LLM API key
- `CARROT_MODEL_PROVIDER` - Model provider (openai, claude)
- `CARROT_MODEL_NAME` - Model name (gpt-4, etc.)
- `CARROT_BASE_URL` - API base URL
